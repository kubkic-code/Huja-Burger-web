# 🍔 Huja Burger - Případová studie (Refaktoring)

Tento repozitář ukazuje můj vývoj a posun ve frontendovém vývoji za poslední 3 roky. Rozhodl jsem se vzít jeden ze svých prvních projektů a kompletně ho od základu přepsat pomocí moderních postupů a technologií.

## 📂 Obsah repozitáře

V tomto repozitáři najdete dvě verze projektu, které demonstrují můj pokrok:

### ❌ [v1-puvodni-verze](./v1-puvodni-verze)
Toto je můj původní kód vytvořený před třemi lety. Reprezentuje mé začátky a dřívější pochopení webového vývoje.
- **Problémy:** Nadměrné používání inline stylů, absolutního pozicování, špatná struktura kódu a chybějící responzivita. Zbytečné použití Flasku pro statický web.

### ✅ [v2-moderni-verze](./v2-moderni-verze)
Toto je nová, kompletně přepsaná verze. Ukazuje mé současné dovednosti.
- **Vylepšení:** 
  - Čisté a sémantické **HTML5** (použití tagů jako `<header>`, `<main>`, `<article>`).
  - Rozvržení postavené na moderním **CSS Gridu** a **Flexboxu**.
  - Plná **responzivita** (Mobile-first přístup).
  - Moderní UI/UX prvky (Glassmorphism, hover animace, CSS proměnné).
  - Odstranění zbytečného backendu, web je čistě statický a bleskově rychlý.
